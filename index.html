Изученные команды

Основные команды для работы

  • git status - узнать текущее состояние репозитория
  • git add "имя файла" - начать отслеживание (проиндексировать если отслеживается) указанного файла (--all вместо имени, чтобы отслеживать/проиндексировать все)
  • git commit - зафиксировать текущие индексируемые изменения в репозитории
    добавить комментарий: -m "текст комментария"
  • git push - отправка текущих изменений в удаленный репозиторий
