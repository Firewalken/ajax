Изученные команды

Основные команды для работы

  • git status - узнать текущее состояние репозитория
  • git add "имя файла" - начать отслеживание (проиндексировать если отслеживается) указанного файла (--all вместо имени, чтобы отслеживать/проиндексировать все)
  • git commit - зафиксировать текущие индексируемые изменения в репозитории
    добавить комментарий: -m "текст комментария"
  • git push - отправка текущих изменений в удаленный репозиторий

Команды для работ с коммитами и веткаами

  • git checkout {name} - передвинуть указатель HEAD на другой коммит или ветку
  • git checkout -b {name} -создать новую ветку {name} и передвинуть указатель на неё
  • git merge {branchname} - влить ветку {branchname} в текущую ветку

Команды для работы с working tree

  • git reset {commit} OR git reset --soft HEAD~ - (перемещение head) состояние до коммита (факлы изменены, добавлены(add) и готовы к коммиту) (редактрование коммита)
  • git reset [--mixed] HEAD~ - (обновление индекса)сосояние до добавления файлов в индекс(add) и до коммита (редактирование индекса)
  • git reset --hard - (обновление рабочей категории) - отменяет все изменения в каталоге
